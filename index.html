Every AI sub claims it's a steal. This board scores them on one brutal question: how much real value do you get per dollar? Each plan gets a Cost-Worth Index (CWI 0–100), ranked against its category. 100 = best value-per-dollar in its class. Updated weekly.

// HOW THE SCORE WORKS

  1. Value per dollar = usable monthly value (quota + access + features) ÷ monthly price.
  2. CWI 0–100 normalizes that within each category — the best-value plan in a class scores 100, the rest scale against it.
  3. Verdict bands: 85+ WORTH IT 65–84 SOLID 45–64 MEH <45 OVERPRICED
  4. CWI measures quantity-per-dollar. Quality, aesthetics and lock-in are captured in each card's best for / weakness notes — a low score isn't "bad", it means you pay a premium per unit.

Figures are curated human estimates of usable value, not scraped invoices. Compare classes, not absolutes. Edit data/services.seed.json and re-run scripts/update_data.py to keep it honest.
