## What is this repo for?
- A 2d dot style, unified ui for : "How much this AI Service/LLM/Subscription worth money?" aka, "Is it worth price?"

## Target
- AI coding services : CursorAI, Copilot, Codex, Claude Code
- AI generation services : NovelAI, Grok Imagine etc...
- AI music services : Suno etc...
- AI research paper services : alpharxiv, Scite.ai

## What we are gonna show?
- The view will be updated weekly. The website(repo/program or whatever..) will fetch informations from those services -> Then analyze moeny<->Service worth status
- For example, Copilot was very cheap and substantially cost worth when use AI LLM calling services. But, recently they changed their services, which now makes 14 times less cost effective in Claude opus usage
- Our goal is to show this kind of changes but base on money. For example, Claude Code 20x costs almost 200 dollars but it has very great features/more quota. Therefore, just for money scale, Claude Code Max 20x is way more cost effective than Claude Code Pro in coding.
