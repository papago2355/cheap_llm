# 💸 IS IT WORTH IT? — AI 서비스 가성비 지수 (발표자료)

> **"이 AI 서비스/구독, 진짜 돈값 하나?"** 를 한 화면에서 답하는 2D 도트 스타일 대시보드

---

## 1. 한 줄 소개

AI 코딩 · 이미지 · 음악 · 연구 구독 서비스를 **"1달러당 실제 가치"** 기준으로 점수화(**CWI 0~100**)하고,
가격을 **매주 공식 가격 페이지에서 자동으로 가져와 검증**하는 픽셀 아트 웹 대시보드.

---

## 2. 문제 (Problem)

- AI 구독은 전부 "싸다 / 가성비 최고"라고 광고하지만, **실제 돈값은 비교가 어렵다.**
- 가격 정책이 **조용히 바뀐다.** 예) GitHub Copilot이 *premium request* 과금으로 전환되며
  프론티어 모델(Opus 등) 사용 가성비가 사실상 **수 배** 나빠짐.
- 사용자는 "Claude Code Max 20x($200)가 Pro($20)보다 비싼데 정말 더 이득인가?" 를
  **숫자로** 판단할 방법이 없다.

## 3. 해결 (Solution)

| 핵심 | 설명 |
|------|------|
| **Cost-Worth Index (CWI 0~100)** | 카테고리 내에서 *1달러당 가치*를 정규화. **100 = 그 부류 최고 가성비.** |
| **주간 자동 갱신** | 매주 월요일 공식 가격 페이지를 fetch → 가격 검증 → 점수 재계산 → 자동 배포 |
| **출처(Provenance) 표기** | 카드마다 `● LIVE / ✓ VERIFIED / ⚠ REVIEW / EST.` 배지 + 출처 링크 + 확인 날짜 |
| **추세 그래프** | 6주 스파크라인으로 "언제 가성비가 나빠졌는지"를 시각화 |

---

## 4. 데모 화면 구성

1. **상단 LIVE API 토큰 가격 띠** — 프론티어 모델의 실시간 1M 토큰 단가 (예: GPT-4o 입력 $2.5 / 출력 $10)
2. **카테고리 탭** — AI 코딩 / 이미지 / 음악 / 연구
3. **서비스 카드** — 가격 + 출처 배지 + Worth Meter(20칸 도트 게이지) + 판정 도장 + 강점/주의 + 6주 추세
4. **정렬** — 가성비순 / 가격순 / 1달러당 가치순
5. **방법론 섹션** — 점수 계산식과 한계 투명 공개

---

## 5. 이번 주 결과 (예시 · 2026-06-22 기준)

| 카테고리 | 1위 (가성비) | CWI | 가격 | 가격 상태 |
|----------|--------------|-----|------|-----------|
| AI 코딩 | Claude Code **Max 20x** | 100 | $200 | ✓ VERIFIED |
| AI 이미지 | NovelAI **Opus** | 100 | $25 | ● LIVE |
| AI 음악 | Suno **Premier** | 100 | $30 | ⚠ REVIEW |
| AI 연구 | Consensus **Premium** | 100 | $9 | ⚠ REVIEW |

> 판정 밴드: `85+ WORTH IT` · `65–84 SOLID` · `45–64 MEH` · `<45 OVERPRICED`

---

## 6. 점수 계산 방식 (Methodology)

```
1달러당 가치 = (월간 실사용 가치: 쿼터 + 모델 접근성 + 기능) ÷ 월 가격
CWI(0~100)  = 같은 카테고리 안에서 1달러당 가치를 정규화 (최고 = 100)
```

- CWI는 **"단위당 양"** 의 가성비를 측정. **품질·미감·락인**은 카드의 *강점/주의* 노트로 보완.
- 즉, 낮은 점수 = "나쁘다"가 아니라 **"단위당 비싸게 주고 산다"** 는 의미 (예: Midjourney = 품질형).

## 7. ⭐ 핵심: 라이브 데이터 파이프라인 (가짜 가정 X)

이전 버전은 가격을 **사람이 가정**했지만, 지금은 **공식 페이지에서 직접 가져온다.**

```
[월요일 06:00 UTC · GitHub Actions]
  fetch_prices.py  →  각 서비스 공식 가격 페이지 fetch
                      ├ 월 가격 자동 추출 (월단위 문맥 + 정상범위 게이트)  → ● LIVE
                      ├ 추출 실패 시 기준가가 페이지에 존재하는지 검증     → ✓ VERIFIED
                      └ 못 찾으면 플래그 (기준가 유지, 사람이 확인)        → ⚠ REVIEW
  update_data.py   →  라이브 가격 병합 + CWI 재계산 + 주간 스냅샷 누적
  git commit/push  →  Vercel 자동 재배포
```

**신뢰성 장치 — 잘못된 값은 절대 덮어쓰지 않는다:**
- 추출은 **"월(month)" 문맥**이 있고 **예상 범위 내**일 때만 채택 → *연 결제 환산가($17/mo billed annually)* 를
  월 가격으로 잘못 publish 하는 전형적 실수를 차단.
- 모든 가격에 **출처 URL + 확인 날짜 + 상태**가 따라붙음 (감사 가능).

**현재 커버리지:** 21개 플랜 중 **9개 자동 LIVE/VERIFIED**, 나머지는 정직하게 REVIEW 플래그.
(Cursor Ultra, ChatGPT Plus/Pro 등은 클라이언트 렌더링이라 정적 HTML에 없음 → Playwright 헤드리스 도입이 업그레이드 경로)

## 8. 기술 스택 / 아키텍처

- **프런트엔드:** 빌드 없는 정적 사이트 (HTML / CSS / Vanilla JS) → Vercel에서 무빌드 배포, 깨질 일 없음
- **데이터 엔진:** Python (`fetch_prices.py`, `update_data.py`) → `js/data.js`(window.DATA) + `data/services.json` 생성
- **자동화:** GitHub Actions 주간 cron → commit → Vercel Git 연동 자동 재배포
- **디자인:** Press Start 2P + VT323 픽셀 폰트, CRT 스캔라인, 도트 Worth Meter

```
데이터 흐름:
sources.json(공식 URL) ─┐
                        ├─ fetch_prices.py ─→ prices.live.json ─┐
services.seed.json(가치)─┘                                      ├─ update_data.py ─→ data.js / services.json ─→ 웹
```
